# sprint1-data-analysis
Projeto desenvolvido durante o bootcamp TripleTen.


## Objetivo

Aplicar conceitos iniciais de análise de dados utilizando Python e Jupyter Notebook, com foco na exploração, organização e interpretação de dados.

## Tecnologias utilizadas

- Python
- Jupyter Notebook
- Pandas

## O que foi feito

- Carregamento e exploração inicial dos dados
- Verificação da estrutura do dataset
- Análise de informações relevantes
- Organização dos resultados em notebook

## Aprendizados

Este projeto ajudou a consolidar os primeiros passos em análise de dados, incluindo leitura de dados, exploração inicial e estruturação de conclusões.
